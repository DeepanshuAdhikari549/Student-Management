# SchoolHub Management System

A simple, fast, and modern full-stack application to manage students and tasks.
## 
Live Link:   https://student-management-jade-seven.vercel.app/
## Tech Stack
- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

## Features
- **Admin Dashboard**: Summary of students and tasks.
- **Student Management**: Full CRUD (Create, Read, Update, Delete) for students.
- **Task Management**: Assign tasks to students, mark them complete, and track progress.
- **Authentication**: Secure admin login using JWT.

## Setup Instructions

### 1. Backend Setup
1. Open `backend/.env` and paste your **MongoDB URI** in `MONGO_URI`.
2. Open a terminal in the `backend` folder.
3. Run `npm install`.
4. Run `npm run dev` to start the server.

### 2. Frontend Setup
1. Open a terminal in the `frontend` folder.
2. Run `npm install`.
3. Run `npm run dev` to start the application.
4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### 3. Initial Admin Login
1. On the login page, click the **"Setup Initial Admin"** button. This will create a default user.
2. Log in with:
   - **Username**: admin
   - **Password**: admin123
